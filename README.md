# Personal Portfolio Website

## About the Project

An interactive and modern personal portfolio built to showcase projects, technical skills, and development experience.

## Features

- Dynamic project sections with detailed descriptions  
- Smooth page and component animations using Motion  
- Interactive 3D laptop model rendered with React Three Fiber  
- Custom 3D model edited and animated in Blender  
- Clean and minimal UI design  

## Tech Stack

- Next.js  
- React  
- TypeScript  
- Tailwind CSS  
- shadcn/ui  
- Motion  
- React Three Fiber  
- Blender  

## Purpose

This project was designed to demonstrate front-end architecture, UI interactions, animation handling, and integration of custom 3D assets within a modern React application.
